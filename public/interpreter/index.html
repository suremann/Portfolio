SYNTAX

  • Number: (? {num} )?
  • Factorial: ({num|var}!)
  • Permutations: (p\i {expression} {expression}) => (P n k)
  • Combinations: (c\i {expression} {expression}) => (C n k)
  • Define a variable: (define\i {identifier} {expression})
  • See all stored variables: (? __env\i__ )?
  • Reset environment and terminal: (? __reset\i__ )?
  • General arithmetic: (? {expression} {[+-*/] {expression}}* )?

    Examples

  • Number: (4), 4
  • Factorial: (4!), (x!), ((1 + 2)!)
  • Permutations: (P 7 3), (P (A) (1 + 2))
  • Combinations: (C 7 3), (C (4!) (A))
  • Define a variable: (define A 23), (DEFINE B (C (4!) (1 + 2)))
  • See all stored variables: __env__
  • Reset environment and terminal: (__reset__)
  • General arithmetic: (1 + 2) * (4!) + (C (1 + 3) (A))

A character followed by `?` is optional.
A character or word followed by `\i` is case insensitive.
Anything followed by a `*` can happen 0 or more times.
Only one element in square brackets can occur at a time.
